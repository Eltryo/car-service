### Description

A RESTful API for car storage.

### Requirements

* [JRE](https://www.java.com/de/download/manual.jsp)

### Installation

Clone the repository and run the following command in the project directory
```sh
java -jar car-rest-api-0.0.1-SNAPSHOT.jar
```
